## インストール方法
1. リポジトリをクローン
   ```bash
   git clone https://github.com/haseatsu114514-dot/uranai_rokkon.git
   cd uranai_rokkon
   ```
2. 依存関係をインストール
   ```bash
   npm install
   # または
   yarn install
   ```
3. 開発サーバ起動
   ```bash
   npm run dev
   # または
   yarn dev
   ```

## 使い方
1. ブラウザで http://localhost:3000 を開く
2. 生年月日や名前を入力して「占う」ボタンを押す
